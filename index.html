Automation Syllabus and Course Overview

Course Overview

This course offers an introduction to Automation Engineering, focusing on hands-on projects to equip you with practical skills.

Learning Outcomes

  • Able to develop and run automated tests
  • Proficient in using GitHub for version control
  • Creating test scripts in markdown format
  • Running tests in a CI/CD pipeline
  • Using Azure DevOps for project management
  • Creating PBIs and understanding Agile methodology
  • And much more!

Tools You'll Be Using

  • IntelliJ IDE for web frameworks
  • Android Studio for Android app development
  • GitHub for version control
  • Microsoft Teams for meetings and messaging

Course Schedule

The course will run for 2 months, with sessions held 3 times a week on Wednesdays, Thursdays, and Fridays from 18:00 to 20:00. If you're unable to join live, all classes will be recorded and made available to you.

Assignments

Starting from the second week, assignments will be given to ensure students remain fully engaged and focused throughout the course.
